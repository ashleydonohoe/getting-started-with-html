Important Concepts

World Wide Web

The World Wide Web was invented in the 1990s and is a collection of HTML documents. Hyperlinks link pages to make the web like an actual web. It can contain plain text, music, video, image, and HTML files. A computer browser makes a request to the Internet, and servers respond with files for browser to display. The HTTP protocol is used by browser to communicate with servers.

HTML

HyperText Markup Language documents are made of the text content you see, markup which is what the content looks like, references to other documents, and links to other pages.

Tags and Elements

Markup is needed to make text look different than plain text. Markup is made up of tags that have a less than symbol, tag name, greater than symbol, tag contents, another less than symbol with a slash, tag name, and last greater than symbol. There's an opening and closing tag (has slash). Two tags with content are known as an element.

The "b" tag is the bold tag, although the "strong" tag can also be used. The "em" tag makes things italicized. The "a" tag is for inserting a link. The "img" tag is for inserting images.

Why Computers Are Stupid

Computers can only understand instructions we humans give them. This means they're sensitive to things like typos and small errors (like a missing end tag).

HTML Attributes

Attributes are used with tags and are in quotes. Tags can have multiple of them. The "a" (or anchor) tag is one example, which is "href" followed by an equal sign and a link in quotes. The "img" tag uses the "src" attribute followed by an equal sign and URL in quotes. The "img" tag also needs an "alt" attribute with alternative text, but it has a void, rather than closing, tag since it has no content.

Whitespace

All whitespace in HTML turns out as a single space. You can use the "br" tag (with void tag) to place a line break. Multiple "br" tags can be used. The "p" (or paragraph tag) also can be used for line breaks. In other programming languages, "\n" is used for line breaks.

Inline vs. Block Elements

The "br", "em", "strong", "b", and "img" tags are inline elements. The "p" tag makes an actual invisible box, so it's a block element.

A span is inline, while a div is block that will create a box around a piece of text. Styles can be attached to them. CSS classes can be used with them.

HTML Document Structure

The structure consists of the doctype, "html", "head" (with "title" tags and links to JS and CSS), and "body" (for actual content of document) opening and closing tags.
